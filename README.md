### 🧩 Xorium Theory

**Xorium Theory** is a new theory of uncertainty. It addresses **Xorness**, a specific
type of uncertainty arising from *indeterminacy, exclusivity, and hesitancy*. To
formalize and quantify Xorness, I introduced the concept of **XOR sets**. When the
universe set is the set of real numbers $\mathbb{R}$, XOR sets specialize to
**XOR numbers**.

> *Xorness emerges from a state of indeterminacy, where multiple mutually exclusive
> possibilities compete for our attention, yet none is strong enough to dominate.*
